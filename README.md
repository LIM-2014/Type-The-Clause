# Type-The-Clause

**Type-The-Clause** is a web-based typing practice project focused on law and constitution texts.  
It allows users to practice typing legal texts, improve typing speed, and memorize content efficiently.

---

## 사용 안내 (Korean)

**Type-The-Clause**는 법률, 헌법 조문을 필사하며 타자 연습을 할 수 있는 웹 기반 프로젝트입니다.  
사용자는 조문을 따라 입력하며 타자 속도를 향상시키고, 법률 내용을 익힐 수 있습니다.

## Features / 기능

- 헌법 및 법률 조문 필사 타자 연습
- 정확도와 속도(WPM) 측정
- 조문 진행률 저장 (브라우저 localStorage)
- 혼자 또는 소규모 그룹으로 학습 가능
